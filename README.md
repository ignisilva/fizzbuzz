# fizzbuzz

Fizzbuzz

3의 배수일땐 'fizz'
5의 배수일땐 'buzz'
15의 배수일땐 'fizzbuzz'
나머지 경우엔 숫자

## 1~100 까지 1씩 증가하는 숫자에 대해 fizzbuzz

## n이 두개 이상의 list, 각 element에 대해 1~100 까지 1씩 증가하는 숫자 fizzbuzz

## trailer comma를 허용하는 언어라면, trailer comma를 사용하는게 좋음
